# 💰 Trip Expense Manager

A lightweight **web application** that helps groups manage and split
travel expenses easily.

The tool allows users to record expenses during a trip and automatically
calculates **who owes whom** at the end of the trip.

The application is deployed using **GitHub Pages**.

------------------------------------------------------------------------

## 🌐 Live Application

https://sourikrishna.github.io/Expense-splitter

------------------------------------------------------------------------

## 📌 About the Project

When traveling with friends or groups, managing expenses can become
confusing.

This project helps groups:

-   Track shared trip expenses
-   Support **equal or unequal contributions**
-   Record **who paid for each expense**
-   Automatically calculate **final settlements**

The system determines **who should pay whom and how much** to settle all
expenses.

## 🧠 Application Workflow

### 1️⃣ Setup Trip

-   Enter number of members
-   Provide names of participants

### 2️⃣ Add Expenses

-   Enter expense description
-   Enter amount
-   Select **who paid**
-   Select **participants involved**
-   Option for **unequal contribution**

### 3️⃣ Expense Tracking

All expenses are stored and displayed in the expense list.

### 4️⃣ Settlement Calculation

The system calculates: - Total balance for each member - Who owes
money - Who should receive money

Final results clearly show **who pays whom**.
